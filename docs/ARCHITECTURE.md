## 🔄 核心流程设计

### 1. 登录流程

```
用户 → 执行 xiaohongshu-login
         ↓
    启动浏览器
         ↓
    打开小红书登录页
         ↓
    显示二维码
         ↓
    用户扫码确认
         ↓
    检测登录成功
         ↓
    提取Cookie
         ↓
    保存到 cookies.json
         ↓
    启动定时刷新任务
         ↓
    返回成功状态
```

### 2. 图文发布流程

```
用户 → "帮我发布小红书图文"
         ↓
    Agent 解析参数
         ↓
    调用 Skill API
         ↓
    发送 HTTP POST /api/v1/publish/image
         ↓
    MCP服务接收请求
         ↓
    加入任务队列
         ↓
    ┌─────────────────┐
    │ 异步处理任务     │
    ├─────────────────┤
    │ 1. 参数校验      │
    │ 2. 图片预处理    │
    │ 3. 浏览器打开    │
    │ 4. 加载Cookie    │
    │ 5. 导航到发布页  │
    │ 6. 填写内容      │
    │ 7. 上传图片      │
    │ 8. 点击发布      │
    │ 9. 等待完成      │
    │ 10. 返回结果     │
    └─────────────────┘
         ↓
    返回发布状态
         ↓
    Agent 通知用户
```

### 3. 视频发布流程

```
用户 → "帮我发布小红书视频"
         ↓
    Agent 解析参数
         ↓
    调用 Skill API
         ↓
    发送 HTTP POST /api/v1/publish/video
         ↓
    MCP服务接收请求
         ↓
    加入任务队列（优先级：高）
         ↓
    ┌─────────────────┐
    │ 异步处理任务     │
    ├─────────────────┤
    │ 1. 参数校验      │
    │ 2. 视频预处理    │
    │    - 格式检查    │
    │    - 大小检查    │
    │ 3. 浏览器打开    │
    │ 4. 加载Cookie    │
    │ 5. 导航到发布页  │
    │ 6. 填写内容      │
    │ 7. 上传视频      │
    │ 8. 等待处理      │
    │    (可能很长)    │
    │ 9. 点击发布      │
    │ 10. 返回结果     │
    └─────────────────┘
         ↓
    WebSocket 推送进度
         ↓
    返回发布状态
```

## ⚙️ 配置管理

### 配置文件结构

```yaml
# config.yaml
server:
  host: "0.0.0.0"
  port: 18060
  mode: "release" # debug, release

auth:
  cookie_file: "./data/cookies.json"
  refresh_interval: "24h"
  max_age: "720h" # 30天

browser:
  headless: true
  timeout: "5m"
  user_agent: "Mozilla/5.0 ..."
  
publish:
  max_retries: 3
  retry_delay: "10s"
  concurrent_limit: 5
  
queue:
  type: "asynq"
  redis:
    addr: "localhost:6379"
    db: 0
  
storage:
  type: "sqlite"
  path: "./data/xiaohongshu.db"
  
log:
  level: "info"
  format: "json"
  output: "./logs/app.log"
  
monitor:
  enabled: true
  port: 9090
```
