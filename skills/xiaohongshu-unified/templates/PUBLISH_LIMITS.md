# 小红书发布限制规则

## 标题限制

### ⚠️ 重要规则

**小红书标题最多20个字！**

### 正确示例

✅ `Claude Code vs Cursor：AI编程助手对比` （17字）
✅ `3分钟搞懂向量数据库！小白也能学会` （16字）
✅ `刚刚！AI圈炸了！Claude发布新功能` （16字）

### 错误示例

❌ `Claude Code还是Cursor？两个AI编程助手深度对比` （21字）
❌ `从零开始学习向量数据库，这一篇就够了！` （21字）

---

## 标签限制

### ⚠️ 重要规则

**每个标签后必须加空格才会变蓝色可点击！**

### 正确格式

```
#AI编程 #编程工具 #效率工具 
```

### 错误格式

```
#AI编程#编程工具#效率工具
```

---

## 正文限制

- **字数**：建议500-1000字
- **分段**：每段≤3行
- **emoji**：每1-2句1个

---

## 配图限制

- **数量**：至少1张，最多9张
- **格式**：JPG、PNG
- **大小**：建议<5MB

---

## 完整检查清单

### 发布前必查

- [ ] 标题 ≤ 20字
- [ ] 标签后有空格
- [ ] 正文500-1000字
- [ ] 至少1张配图
- [ ] 地点已填写
