# 小红书发布规则

## 标签格式规则

### ⚠️ 重要

**标签必须加空格才会变蓝色可点击！**

### 正确格式

```
#AI编程 #编程工具 #效率工具 #程序员日常 #开发者 #AI助手 
```

### 错误格式

```
#AI编程#编程工具#效率工具
```

---

## 标准输出格式

```
一、标题
```
[标题内容]
```

二、正文
```
[正文内容]

#标签1 #标签2 #标签3 
```
⚠️ **注意**：每个标签后必须加空格

三、配图
[图片提示词或URL]

四、地点
```
北京
```
```
