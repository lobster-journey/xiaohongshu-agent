## 使用流程

### 标准流程

```
热点收集 → 内容生成 → 文案优化 → 配图生成 → 交付用户 → 用户手动发布
```

### 快速生成

当用户要求生成小红书笔记时：

**步骤1**：确定主题
- 自动从热点中选择
- 或使用用户指定主题

**步骤2**：生成内容
```bash
# 标题公式（选择最适合的）
- 数字+痛点：3个方法让你告别XX
- 疑问+好奇：为什么XX？秘密都在这
- 对比+反差：从XX到YY，我只做对了这件事
- 警告+紧迫：千万别再XX了！后果很严重
- 种草+推荐：亲测好用！这个XX必须安利
```

**步骤3**：优化文案
- 添加emoji（每1-2句1个）
- 分段（每段≤3行）
- 符号突出（「」【】）
- 语气词（呀、呢、啦）
- 标签（2-5个）

**步骤4**：输出标准格式

```
一、标题
```
[标题内容]
```

二、正文
```
[正文内容]

#标签1 #标签2 #标签3
```

三、配图
[即梦提示词]

四、地点
```
北京
```
```

## 配图生成

### 即梦API集成

**前置条件**：
- Session ID有效
- 积分充足（>800分）

**生成流程**：
```bash
# 调用即梦API
curl -X POST http://localhost:5100/v1/images/generations \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $SESSION_ID" \
  -d '{
    "prompt": "图片描述",
    "ratio": "16:9",
    "resolution": "2k"
  }'
```

**成本控制**：
- 生图：3分/次
- 生视频：210分/次
- 预警：<800分
